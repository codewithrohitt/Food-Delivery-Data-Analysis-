# 🍽️ Food Delivery Data Analysis Project

## 📌 Overview

This project analyzes a food delivery dataset containing ~197K orders to extract actionable business insights related to sales performance, customer behavior, and regional trends.

---

## 🔧 Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Plotly

---

## 📊 Key KPIs

* Total Sales: ₹5.3 Cr
* Total Orders: 197K+
* Average Order Value: ₹268
* Average Rating: 4.3
* Total Ratings Count: 5.5M+

---

## 📈 Analysis Performed

* Monthly, Weekly, and Daily Sales Trends
* Veg vs Non-Veg Revenue Comparison
* State-wise Revenue Distribution
* Quarterly Performance Summary
* Top 5 Cities by Sales

---

## 🔍 Key Insights

* Strong revenue generation with consistent order volume
* Higher sales observed during weekends
* Non-Veg items dominate revenue contribution
* Few cities contribute significantly to total revenue
* Seasonal variation observed in quarterly performance

---

## ⚠️ Limitations

* No unique Order ID available
* Rating data is highly skewed (many zero values)
* Food classification based on keyword matching
* No profitability data available

---

## 💡 Conclusion

This project demonstrates how raw data can be transformed into meaningful insights to support business decision-making in the food delivery domain.
